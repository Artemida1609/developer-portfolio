# Project Images

Place your project screenshots in this directory with the following filenames:

- `gadget-store.png` - Screenshot for Gadget Store project
- `2048-game.png` - Screenshot for 2048 Game project  
- `todo-app.png` - Screenshot for Todo App project

## Image Requirements

- **Format**: PNG or JPG
- **Recommended size**: 800x600px or 16:9 aspect ratio
- **File size**: Optimize images to keep file size under 500KB for better performance

The images will be automatically displayed in the project cards on the portfolio website.
